Proyek ini adalah API backend yang dibangun menggunakan FastAPI dan SQLAlchemy ORM sebagai bagian dari latihan Pemrograman 3. Aplikasi ini dirancang untuk mengelola sistem toko online sederhana yang mencakup manajemen data customers, products, dan orders. Proyek ini memfokuskan implementasi operasi CRUD yang efisien, validasi data menggunakan Pydantic, serta teknik analisis data tingkat lanjut seperti penggunaan JOIN, Aggregation, Subquery, CTE, dan CASE statement dalam SQLAlchemy tanpa menggunakan fitur relationship().
